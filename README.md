## 軟體目的

1. 對一般使用者，這是一個可以簡單上傳到網路空間就能運行的軟體。讓使用者可以分享自己的檔案。
2. 對開發者而言，這是一個小型的檔案櫃模型，提供前端與後端的接口。

## 使用說明

1. 開啟 Sever 資料夾中的 pwdSetting.html，依指示更改密碼(預設密碼為空字串)。
2. 將 Sever 整個資料夾的檔案複製到支持 PHP 的網頁伺服器空間即完成。
